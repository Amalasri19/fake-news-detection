 Fake News Detection using Machine Learning

 📌 Project Description
This project detects whether a news article is **real or fake** using Machine Learning and Natural Language Processing (NLP).  
The model analyzes the text of news articles and predicts their authenticity.

Fake news detection helps prevent the spread of misinformation on the internet.
🚀 Features
- Detects fake and real news
- Uses Machine Learning algorithms
- Text preprocessing and NLP techniques
- Easy to run and test
🛠 Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
📂 Project Structure
- trainmodule.py
- app.py
- templates
    1.index.html
    2.result.html
- fake.csv
- true.csv
- model.pkl
- prediction_log.txt
- vectorizer.pkl
- style.css
📊 Output
The model predicts whether the given news is:
  Real News
  Fake News
